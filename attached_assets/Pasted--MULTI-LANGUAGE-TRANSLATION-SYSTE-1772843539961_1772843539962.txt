IMPLEMENTATION:

Translation Architecture:
✓ Use i18next or react-intl for frontend translations
✓ Store translations in JSON files per language
✓ Dynamic language switching without page reload
✓ Persist language preference (localStorage + database)
✓ Auto-detect browser language on first visit
✓ Language selector in header/settings

VTX AI Multi-Language Training:
✓ VTX must detect user's selected language
✓ Respond in same language user is using
✓ Understand crypto terminology in all languages
✓ Maintain context across language switches
✓ Translate technical terms accurately
✓ Handle mixed-language inputs gracefully

System Prompt for VTX with Language Awareness:
"You are VTX AI, an intelligent crypto assistant for Naka Labs. The user's current language is: [DETECTED_LANGUAGE]. You must:
1. Always respond in [DETECTED_LANGUAGE]
2. Understand crypto terminology in this language
3. Provide accurate translations of technical terms
4. Maintain professional tone in all languages
5. If user asks something in different language, respond in that language
6. For wallet addresses, token symbols, and blockchain data, keep original format (no translation)"
