## 9.1 ANTHROPIC (Claude AI)

**Used In:** lib/services/anthropic.ts — all AI features
**Current Integration:** claude-sonnet-4-6 (executor) + claude-opus-4-6 (advisor via beta header)
**Auth:** ANTHROPIC_API_KEY (Bearer token)

**Pricing (as of 2026):**
- claude-sonnet-4-6: $3/MTok input, $15/MTok output
- claude-opus-4-6: $15/MTok input, $75/MTok output (advisor uses this)
- Advisor strategy ~80-90% cost saving vs full Opus per request

**Rate Limits:** Based on API tier — default limits increase with usage history
**Context Window:** 200,000 tokens (claude-sonnet-4-6 and claude-opus-4-6)
**Max Output Tokens:** 8,192 by default (platform uses 4,096)

**Current Usage Pattern:**
- VTX Agent: tool loop up to 5 iterations, each calling Sonnet + up to 2x Opus advisor
- DNA Analyzer: vtxAnalyze() — single call, 1,500 tokens max
- Shadow Guardian: vtxAnalyze() — single call, 200 tokens max
- Sniper Execute: vtxAnalyze() — single call, 200 tokens max
- Contract Analyzer: vtxAnalyze() — single call, 280 tokens max

**Assessment:** Correct and cost-efficient. Advisor strategy is smart. No issues.

## 9.2 HELIUS (Solana)

**Used In:** lib/services/helius.ts, lib/agents/, app/api/wallet-clusters, app/api/network-graph, app/api/mev-protection, app/api/context-feed
**Auth:** HELIUS_API_KEY_1 and HELIUS_API_KEY_2 (query param or header, 2-key rotation)

**APIs Used:**
- `getTokenAccountsByOwner` — Solana RPC
- `getBalance` — Solana RPC
- `getTokenSupply` — Solana RPC
- `getTokenLargestAccounts` — Solana RPC
- `/token-metadata` (Helius Enhanced API) — batch token metadata
- `/addresses/{address}/transactions` (Helius Enhanced API) — enhanced tx history

**Free Tier:** 1M credits/month (each enhanced tx = ~1 credit; raw RPC calls = smaller fractions)
**Developer Plan:** $49/month — 30M credits, 50 req/sec
**Business Plan:** $499/month — 500M credits, 500 req/sec
**Enterprise:** Custom

**Current Usage Risk:** Two-key rotation doubles effective free tier. At scale (1000+ active users), the wallet intelligence pipeline alone (parallel Helius calls per wallet analysis) will exhaust free tier rapidly.
**Recommendation:** Developer Plan ($49/mo) at minimum for production. Monitor credit consumption from day one.

## 9.3 BIRDEYE (Solana)

**Used In:** lib/services/birdeye.ts — Solana prices, wallet tokens, top traders, OHLCV, holders, trending, new listings
**Auth:** BIRDEYE_API_KEY (header: X-API-KEY)

**APIs Used:**
- `/defi/price` — single token price
- `/defi/multi_price` — batch up to 100 tokens
- `/defi/token_overview` — full token overview
- `/defi/token_holders` — holder list
- `/defi/ohlcv` — OHLCV data
- `/defi/token_list` — top tokens
- `/defi/wallet/token_list` — wallet holdings (expensive)
- `/defi/wallet/transaction_list` — wallet tx history (expensive)
- `/trader/gainers-losers` — top traders PnL

**Free Tier:** Limited — 1 req/sec, limited endpoints
**Starter Plan:** ~$50/month — most endpoints, higher rate limits
**Professional Plan:** ~$200/month — all endpoints, higher limits
**Business Plan:** ~$500/month

**Critical:** `getWalletTokenList()` and `getWalletTransactions()` are marked as expensive endpoints in Birdeye pricing. These are used in every wallet intelligence request and DNA analyzer pipeline. Free tier will exhaust quickly.
**Recommendation:** Professional Plan ($200/mo) for production.

## 9.4 COINGECKO

**Used In:** lib/services/coingecko.ts — market data, trending, global stats, OHLCV
**Auth:** COINGECKO_API_KEY (query param: x_cg_pro_api_key)
**Fallback:** Public API on 429 error

**Current Setup:** Pro API key with public fallback on rate limit errors. Well-implemented.

**Free (Public) API:** 30 calls/min, limited endpoints, no 1h price changes
**Pro API:** ~$129/month — 500 calls/min, 1h price change data, full OHLCV history
**Analyst API:** ~$499/month — 2,000 calls/min

**Why Pro Matters:** Notifications route needs `price_change_percentage_1h_in_currency` which requires Pro API. Portfolio route needs OHLCV history for charts.
**Recommendation:** Pro API required for production — public fallback is insufficient at scale.
